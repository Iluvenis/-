# 第十章作业

1. 高程控制测量的主要方法有哪些？各有何优缺点？
   答：水准测量、高程控制网、GPS高程控制测量
   1. 水准测量精度高，但难以在地形崎岖的地区进行
   2. 高程控制网便于建立，但精度低
   3. GPS高程控制测量需要相关基础设施

2. 水准测量路线的布设形式有哪些？
   答：单一水准路线和水准网。单一水准路线又可分为附合水准路线、闭合水准路线和支水准路线。
